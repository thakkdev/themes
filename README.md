# Website Themes Collection

A collection of modern, responsive, and ready-to-use website themes for various purposes. Each theme is self-contained with its own folder, assets, and documentation.

## Available Themes

### 1. Basic E-Commerce Theme
A clean and modern static e-commerce website theme with shopping cart functionality.

**Features:**
- Responsive design for all devices
- Shopping cart with localStorage persistence
- Product catalog with filtering and sorting
- Home, Products, and Cart pages
- Modern UI with smooth animations

**Location:** `basic-ecommerce-theme/`

[View Documentation](basic-ecommerce-theme/README.md)

## Getting Started

1. Browse the available themes in their respective folders
2. Each theme folder contains:
   - HTML files
   - CSS stylesheets
   - JavaScript files
   - README.md with detailed documentation
   - Assets folder for images and other resources

3. To use a theme:
   - Navigate to the theme folder
   - Open `index.html` in your browser
   - Follow the theme's README for customization instructions

## Theme Structure

Each theme follows this standard structure:
```
theme-name/
├── index.html          # Main page
├── [other-pages].html  # Additional pages
├── css/                # Stylesheets
├── js/                 # JavaScript files
├── images/             # Image assets
├── assets/             # Other assets
└── README.md           # Theme documentation
```
